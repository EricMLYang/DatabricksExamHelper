## 題目內容

### 題幹

A production cluster has 3 executor nodes and uses the same virtual machine type for the driver and executor.

When evaluating the Ganglia Metrics for this cluster, which indicator would signal a bottleneck caused by code executing on the driver?

### 選項

- **A.** The five Minute Load Average remains consistent/flat
- **B.** Bytes Received never exceeds 80 million bytes per second
- **C.** Total Disk Space remains constant
- **D.** Network I/O never spikes
- **E.** Overall cluster CPU utilization is around 25%

## 答案與解析連結

### 正確答案
**正解:** `D`

### 解析檔案
**詳細解析:** [點此查看解析](../analysis/Q-050-analysis.md)

**備註:** Community discussion is split between D and E. E implies underutilization of the cluster which usually happens when the driver is the bottleneck.
